HTML writing rules

기본규칙

  1. HTML은 해당 DTD의 명세에 맞게 작성하며, W3C validation을 통과해야 한다.
  2. 특정 요소에 class, style을 선언할 때는 선언 순서를 준수한다. 다음과 같이 class와 style은 제일 뒷부분에 선언한다. <input type="text" id="user_id" title="사용자ID" class="input_txt" style="width:100px">
  3. DTD(Document Type Definition)를 제외한 모든 요소와 애트리뷰트는 소문자로 작성한다.
  4. 애트리뷰트값은 큰따옴표(" ")로 묶는다.
  5. 특수 기호는 문자 엔티티 참조를 사용하여 코드로 변환한다.

주석표기

  1. 주석 기호와 주석 내용 사이에는 반드시 공백 한 칸이 있어야 한다.
  2. 시작과 종료 주석의 주석 내용은 동일해야 한다. <!-- 주석 내용 -- > <!-- // 주석 내용 -- >
  3. HTML 코드와 주석은 줄로 분리해야 한다.
  4. 개발 적용과 관련된 주석은 해당되는 영역 위에 표기하며, 종료 주석은 표기하지 않는다. 주석 앞에는 [D]라는 말머리를 사용하여 담당 개발자가 반드시 확인할 수 있도록 한다.
