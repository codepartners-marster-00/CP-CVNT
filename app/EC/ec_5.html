웹 접근성을 준수한 Tab 사용 규칙

  1. <code> 에 role="tablist" 속성을 사용하여 탭 목록임을 명시한다.
  2. <li>요소를 사용하여 각 탭 항목을 정의하고, role="presentation" 속성을 사용하여 목록의 항목이 탭이라는 것을 표시한다.
  3. 버튼을 그룹핑 할 경우 role="group" 을 사용하여 그룹을 명시하는 것을 권장한다.
  4. <a> 요소를 사용하여 탭 링크를 정의하고, role="tab" 속성을 사용하여 링크가 탭 역할을 수행함을 표시한다.
  5. aria-controls 속성을 사용하여 해당 탭이 제어하는 콘텐츠의 ID를 지정하고, aria-selected 속성을 사용하여 현재 선택된 탭을 표시한다.
  6. <div> 요소를 사용하여 탭 콘텐츠를 정의하고, role="tabpanel" 속성을 사용하여 콘텐츠의 역할을 자정한다.
  7. 각 탭 콘텐츠에는 aria-labelledby 속성을 사용하여 해당 탭의 ID를 참조하여 탭과 콘텐츠의 관계를 지정한다.
